Теперь буквально пару слов про историю создания спецификации и ее наименования.

В декабре 1995 LiveScript был переименован в JavaScript и интегрирован в браузер Netscape Communicator.

С самого начала JavaScript показал отличные возможности, поэтому другие браузеры также были вынуждены искать подобные решения. Например, Microsoft разработала свою реализацию JavaScript, назвав ее JScript. Первая версия JScript появилась в Internet Explorer 3.0 в августе 1996 года.

Первой большой переменой для JavaScript после его выпуска стала стандартизация ECMA. ECMA — ассоциация, созданная в 1961 году с целью стандартизации информационных и коммуникационных систем.

Работа над стандартизацией JavaScript началась в ноябре 1996 года. Стандарту был присвоен идентификационный номер ECMA-262.

Из-за проблем с торговой маркой ECMA не могла использовать JavaScript в качестве названия, поэтому было решено назвать описанный стандартом язык программирования ECMAScript. На сегодняшний день JavaScript это всего лишь коммерческое название ECMAScript.

Первый стандарт ECMAScript был основан на версии JavaScript, входившей в состав Netscape Navigator 4 и был выпущен в июне 1997 года.

ECMAScript 2, была выпущена в июне 1998 года, чтобы исправить несостыковки между ECMA и стандартом ISO для JavaScript (ISO/IEC 16262).

ECMAScript 3 был выпущен в декабре 1999 года. В него вошли такие крупные новшества, как регулярные выражения, блок do-while, исключения и try/catch блоки и другие улучшения.

В последующие годы шла параллельная работа над стандартами 3.1 и 4. В комитеты по разработке входили Adobe, Mozilla, Opera, Microsoft и Yahoo.

Кстати, на ранней версии ECMAScript 4 был основан язык ActionScript. Adobe включил его в состав пакета инструментов для Flash-разработки.

В 2009 году ECMAScript 3.1 был полностью завершён и одобрен всеми участниками комитета. Так как ECMAScript 4 считался незаконченным, было решено переименовать ECMAScript 3.1 в ECMAScript 5, чтобы избежать недоразумений.

В 2015 году вышел ECMAScript 6, позже переименованный в ECMAScript 2015, в него вошли некоторые из основных возможностей ECMAScript 4.

Начиная с 2015 года обновления спецификации выходят каждый год, и получают соответствующее наименование:
ЕСМА-262, 7th edition - ECMAScript 2016
ЕСМА-262, 8th edition - ECMAScript 2017
...
ЕСМА-262, 14th edition - ECMAScript 2023

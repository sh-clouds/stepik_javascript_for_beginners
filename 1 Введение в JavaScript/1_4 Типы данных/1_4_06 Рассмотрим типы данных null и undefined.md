Рассмотрим типы данных null и undefined.

Тип данных null не относится ни к одному из рассмотренных ранее типов. Это отдельный тип данных, который может содержать только значение null.

let x = null;
Значение null является примитивным типом, а не объектом и не ссылкой на несуществующий объект, в отличие от некоторых других языков программирования. 

В отличие от значения undefined, которое обозначает неопределенность значения, null обозначает определенное отсутствие значения.


Undefined также является отдельным типом, которое может принимать только одно значение - undefined. Это означает, что значение не было присвоено, например в том случае, когда переменная была только объявлена, но не проинициализирована. Любая объявленная переменная до ее инициализации, т.е. первичного присвоения значения, имеет тип undefined.

let x; //Значение переменной х в настоящий момент - undefined
Говоря об этих типах данных, необходимо упомянуть различия при использовании операторов сравнения (сами операторы сравнения более подробно будут рассмотрены в следующем уроке), поскольку результат сравнения null и undefined будет разный, в зависимости от используемого оператора. Если используется оператор равенства (==), то результат сравнения будет положительным, а если оператор идентичности (===), то отрицательным.

let x = null;     // the value of the variable x is null
let y;            // the value of the variable y is undefined

let z1 = x == y;  // the comparison result will be true
let z2 = x === y; // the comparison result will be false

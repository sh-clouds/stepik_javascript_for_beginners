Явное преобразование к численному типу производится функцией Number().
Если аргументом этой функции является строка, или переменная, содержащая строку, то при условии, что в строке есть только цифры, функция вернет численное значение. Для проверки типа данных воспользуемся функцией typeof, упомянутой в прошлых шагах. Например:

let x = "2"            // Мы создали переменную х и поместили в нее строковое значение "2"
console.log(typeof x)  // Вывели в консоль тип данных, там будет написано - string
let y = Number(x)      // Переменной y присваиваем значение переменной x, преобразованное в число
console.log(typeof y)  // Проверяем тип данных в переменной y - там будет number
Если в переменной, которую мы пытаемся преобразовать в число, есть любые другие символы, кроме цифр, преобразование не получится, и в результат будет записано значение NaN (Not a Number), знакомое нам по прошлым шагам. Также, если значение переменной, которую мы пытаемся преобразовать в число, будет Undefined, то преобразование тоже не удастся, и результатом выполнения операции также будет NaN.

Однако, если в переменной будет значение null, то оно вполне нормально будет преобразовано в число - результатом будет 0 (ноль).
Также результатом будет 0, если мы попытаемся преобразовать пустую строку.

let x = "";
console.log(Number(x));  // В результате выполнения этой команды в консоль будет выведено значение 0
Еще один важный момент - при преобразовании у строки удаляются пробелы в начале и в конце строки. Т.е. строка " 123 " будет преобразована в число 123. Однако, если пробелы будут в середине, то результатом преобразования будет NaN.

let x = " 123 4";
console.log(Number(x));  // В результате выполнения этой команды в консоль будет выведено значение NaN
В число можно преобразовывать не только строковые, но и логические данные. Здесь правило простое - true превращается в 1, а false превращается в 0.

let x = false;
console.log(Number(x));  // В результате выполнения этой команды в консоль будет выведено значение 0

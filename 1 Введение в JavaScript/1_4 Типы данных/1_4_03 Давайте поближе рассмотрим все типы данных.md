Давайте поближе рассмотрим все типы данных, начиная с числового.

Максимальные значения, которые мы можем поместить в переменную числового типа, лежат в диапазоне от -(2**53-1) до (2**53-1), т.е. от -9007199254740991 до 9007199254740991.

При необходимости использовать бОльшие значения, мы можем создать переменную типа BigInt - для этого при инициализации переменной необходимо добавить в конце значение знак "n":

let veryLongInteger = 12345678901234567890n;
Числовой тип данных может представлять собой как целые, так и дробные числа.

let width = 100;
let height = 200.2;
Мы можем менять значение переменной произвольным образом между целочисленным и дробным значением:

let height = 200;
height = 200.005;
К числовому типу данных также относятся особые числовые значения - бесконечность, отрицательная бесконечность и "не число" - Infinity, -Infinity и NaN.


Infinity. Бесконечность это значение, которое больше любого числа. Например, мы можем получить его в результате деления на ноль либо задать его явно.

x = 2/0;
x = Infinity;
В обоих случаях значение переменной x будет Infinity.


- Infinity. Примерно такая же история с минус бесконечностью.

x = 1 - Infinity;
x = - Infinity;

C NaN все немного сложнее.
NaN означает Not A Number и, фактически, являеся результатом любой некорректной вычислительной операции. Любая вычислительная операция с не-числом будет иметь результатом NaN. Например:

x = 1 / "text"
В результате выполнения этой операции в переменной x окажется значение NaN.

В принципе, в результате выполнения любой вычислительной операции, в которой принимает участие значение NaN, результат будет тоже NaN, кроме одного исключения - математика считает, что любое значение в нулевой степени будет равно единице, поэтому NaN в нулевой степени тоже будет равен единице.

x = NaN**0;
В результате такой операции в переменной x окажется значение 1.

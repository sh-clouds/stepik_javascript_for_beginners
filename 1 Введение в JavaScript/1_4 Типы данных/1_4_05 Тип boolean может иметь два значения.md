Тип boolean может иметь два значения: true и false.

Этот тип данных можно использовать для обозначения истинности или ложности определенного состояния.

let isCheckboxSelected = true;
let doorIsOpen = false;
Такие значения могут быть результатом операции сравнения:

let x = 5;
let y = 3;
let z = x > y;
В результате такого вычисления значением переменной z станет true (операции сравнения, как и другие логические операции, будут рассмотрены более подробно в последующих уроках).

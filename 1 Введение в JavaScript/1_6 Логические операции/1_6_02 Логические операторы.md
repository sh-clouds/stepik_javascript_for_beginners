Логические операторы.

Логический оператор И (&&) (конъюнкция) для набора логических операндов будет истинным тогда и только тогда, когда все операнды истинны. В противном случае оно будет ложным. В более общем смысле, оператор возвращает значение первого ложного операнда, обнаруженного при вычислении слева направо, или значение последнего операнда, если все они верны.
Логическое И (&&) вычисляет операнды слева направо, немедленно возвращая значение первого встреченного ложного операнда; если все значения верны, возвращается значение последнего операнда.

(Если значение можно преобразовать в истину, это значение называется истинным. Если значение можно преобразовать в ложь, это значение называется ложным.)

const a = 3;
const b = -2;

console.log(a > 0 && b > 0); //вывод в консоли будет - false

Логический оператор НЕ (!) (отрицание) переводит истину в ложь и наоборот. Обычно он используется с логическими значениями. При использовании с не логическими значениями он возвращает false, если его единственный операнд можно преобразовать в true; в противном случае возвращается true.

const a = 3;
const b = -2;

console.log(!(a > 0 || b > 0)); // вывод в консоли будет -  false

Логический оператор ИЛИ (||) (дизъюнкция) для набора операндов истинен тогда и только тогда, когда один или несколько его операндов истинны. Обычно он используется с логическими значениями. Если это так, он возвращает логическое значение. Однако || оператор фактически возвращает значение одного из указанных операндов, поэтому, если этот оператор используется с не логическими значениями, он вернет не логическое значение. Если x можно преобразовать в true, возвращает x; иначе возвращает y.

const a = 3;
const b = -2;

console.log(a > 0 || b > 0); // вывод в консоли будет - true

Оператор нулевого слияния (??) — это логический оператор, который возвращает свой правый операнд, если его левый операнд имеет значение NULL или Undefined, и в противном случае возвращает свой левый операнд.

​​​​​​​const a = null ?? 'text string'; 
console.log(a);    // вывод в консоли будет - "text string" 

const b = 0 ?? 42; 
console.log(b);    // вывод в консоли будет - 0

Логические присваивания:

Логический оператор присваивания И (&&=) вычисляет только правый операнд и присваивает значение левому, если левый операнд является истинным. 

let a = 1;
let b = 0;

a &&= 2;
console.log(a);   // вывод в консоли будет - 2

b &&= 2;
console.log(b);   // вывод в консоли будет - 0
Логический оператор присваивания ИЛИ (||=) вычисляет только правый операнд и присваивает значение левому, если левый операнд является ложным. Логическое присваивание ИЛИ замыкает, что означает, что x ||= y эквивалентно x || (x = y), за исключением того, что выражение x вычисляется только один раз.

let a = 1;
let b = 0;

a ||= 2;
console.log(a);   // вывод в консоли будет - 1

b ||= 2;
console.log(b);   // вывод в консоли будет - 2

Оператор нулевого слияния с присваиванием (??=), или оператор логического присваивания с нулевым значением, оценивает только правый операнд и присваивает значение левому, если левый операнд имеет нулевое значение (NULL или Undefined).

let a = null;
a ??= 'text string';
console.log(a);    // вывод в консоли будет - "text string"

let b = 0;
b ??= 42;
console.log(b);    // вывод в консоли будет - 0

Какие операнды являются истинными, а какие ложными. Ложные: false, null, undefined, NaN, пустая строка и 0. Остальные значения - истинные.

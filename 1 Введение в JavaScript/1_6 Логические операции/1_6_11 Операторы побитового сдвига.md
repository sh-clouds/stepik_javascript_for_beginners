Здесь мы видим операторы побитового сдвига:

Оператор сдвига влево (<<) возвращает число или BigInt, двоичное представление которого представляет собой первый операнд, сдвинутый на указанное количество бит влево. Лишние биты, сдвинутые влево, отбрасываются, а нулевые биты сдвигаются справа.

const a = 5;         // 00000000000000000000000000000101
const b = 2;         // 00000000000000000000000000000010

console.log(a << b); // 00000000000000000000000000010100
                     // вывод в консоли будет - 20

Оператор сдвига вправо (>>) возвращает число или BigInt, двоичное представление которого представляет собой первый операнд, сдвинутый на указанное количество бит вправо. Лишние биты, сдвинутые вправо, отбрасываются, а копии самого левого бита сдвигаются слева. Эту операцию еще называют «сдвигом вправо с распространением знака» или «арифметическим сдвигом вправо», поскольку знак результирующего числа совпадает со знаком первого операнда.

const a = 5;         //  00000000000000000000000000000101
const b = 2;         //  00000000000000000000000000000010
const c = -5;        //  11111111111111111111111111111011

console.log(a >> b); //  00000000000000000000000000000001
                     //  вывод в консоли будет - 1

console.log(c >> b); //  11111111111111111111111111111110
                     //  вывод в консоли будет - "-2"

Оператор беззнакового сдвига вправо (>>>) возвращает число, двоичное представление которого представляет собой первый операнд, сдвинутый на указанное количество бит вправо. Лишние биты, сдвинутые вправо, отбрасываются, а нулевые биты сдвигаются слева. Эту операцию также называют «сдвигом вправо с заполнением нулями», поскольку знаковый бит становится 0, поэтому результирующее число всегда положительное. Беззнаковый сдвиг вправо не принимает значения BigInt.

const a = 5;          //  00000000000000000000000000000101
const b = 2;          //  00000000000000000000000000000010
const c = -5;         //  11111111111111111111111111111011

console.log(a >>> b); //  00000000000000000000000000000001
                      //  вывод в консоли будет - 1

console.log(c >>> b); //  00111111111111111111111111111110
                      //  вывод в консоли будет - 1073741822

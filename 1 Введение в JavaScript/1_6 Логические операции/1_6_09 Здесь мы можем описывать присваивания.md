Здесь мы можем описывать присваивания с помощью побитовых операторов.

Оператор побитового присваивания И (&=) выполняет побитовое И для двух операндов и присваивает результат левому операнду.

x &= y эквивалентно x = x & y, за исключением того, что выражение x вычисляется только один раз.

let a = 5;      // 00000000000000000000000000000101
a &= 3;         // 00000000000000000000000000000011

console.log(a); // 00000000000000000000000000000001
                // вывод в консоли будет - 1

Оператор побитового присваивания ИЛИ (|=) выполняет побитовое ИЛИ над двумя операндами и присваивает результат левому операнду.

x |= y эквивалентно x = x | y, за исключением того, что выражение x вычисляется только один раз.

let a = 5;      // 00000000000000000000000000000101
a |= 3;         // 00000000000000000000000000000011

console.log(a); // 00000000000000000000000000000111
                // вывод в консоли будет - 7
Оператор присваивания побитового исключающего ИЛИ (^=) выполняет побитовое исключающее ИЛИ для двух операндов и присваивает результат левому операнду.

x ^= y эквивалентно x = x ^ y, за исключением того, что выражение x вычисляется только один раз.

let a = 5;      // 00000000000000000000000000000101
a ^= 3;         // 00000000000000000000000000000011

console.log(a); // 00000000000000000000000000000110
                // вывод в консоли будет - 6

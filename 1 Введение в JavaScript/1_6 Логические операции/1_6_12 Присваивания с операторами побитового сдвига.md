У нас также есть несколько присваиваний с операторами побитового сдвига:

Оператор присваивания сдвига влево (<<=) выполняет сдвиг влево для двух операндов и присваивает результат левому операнду.

x <<= y эквивалентно x = x << y, за исключением того, что выражение x вычисляется только один раз.

let a = 5;       // 00000000000000000000000000000101

a <<= 2;         // 00000000000000000000000000010100

console.log(a);  // вывод в консоли будет - 20

Оператор присваивания сдвига вправо (>>=) выполняет сдвиг вправо для двух операндов и присваивает результат левому операнду.
x >>= y эквивалентно x = x >> y, за исключением того, что выражение x вычисляется только один раз.

 

let a = 5;       //  00000000000000000000000000000101

a >>= 2;         //  00000000000000000000000000000001
console.log(a);  //  вывод в консоли будет - 1

let b = -5;      //  11111111111111111111111111111011

b >>= 2;         //  11111111111111111111111111111110
console.log(b);  //  вывод в консоли будет - "-2"

Оператор присваивания беззнакового сдвига вправо (>>>=) выполняет беззнаковый сдвиг вправо для двух операндов и присваивает результат левому операнду. x >>>= y эквивалентно x = x >>> y, за исключением того, что выражение x вычисляется только один раз.

let a = 5;       //  00000000000000000000000000000101

a >>>= 2;        //  00000000000000000000000000000001
console.log(a);  //  вывод в консоли будет - 1

let b = -5;      //  -00000000000000000000000000000101

b >>>= 2;        //  00111111111111111111111111111110
console.log(b);  //  вывод в консоли будет - 1073741822

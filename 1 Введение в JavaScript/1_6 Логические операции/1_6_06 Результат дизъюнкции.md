В этой задаче вам нужно вернуть результат дизъюнкции (операции логическое ИЛИ, описанной во втором шаге этого урока) переменной a с переменной b. 

Пример ввода и вывода, показанные ниже, являются примером. Они показывают, что для значений переменных «a» и «b», как в примере ввода, например 7 и 2, ваш результат должен быть таким, как указано в примере вывода. Например, если ввод выборки равен 7 и 2, то вывод должен быть 7.

Sample Input:
7 2

Sample Output:
7

Напишите программу. Тестируется через stdin → stdout
Верно решили 778 учащихся

function testLogical(a, b) {
    let x = a || b;
    return x;

Зачастую нужно создать переменную, значение которой не планируется изменять в процессе выполнения программы. Например длинный текстовый литерал, цифровое обозначение цвета или тому подобные значения. В таком случае удобнее использовать не переменную, а константу.

Принципы работы с константой схожи работе с переменной, за исключением того, что константу нельзя объявить и не инициализировать, т.е. при объявлении константы необходимо сразу присвоить ей значение.

Объявление константы происходит с использованием ключевого слова const. Например:

const contractor = "Plunder and Flee Inc.";
const height = 100;
const width = 200;
const colour = "#FF00FF";

Последний из операторов, который мы рассмотрим, это оператор запятая ",".
Фактически, этот оператор, который позволяет выполнить несколько действий, через запятую, по-очереди.

Порядок выполнения действий, перечисленных через запятую - слева направо. 
Также нужно учесть, что в результате операции возвращается только результат последнего выражения, остальные только выполняются.

let x = 10; 
let y = (x = x + 10,  12 + 13);
console.log(x, y);               // Значение вывода будет "20 25"
В данном примере сначала будет выполнена операция увеличения переменной x на 10, после этого операция сложения чисел 12 и 13, и результат именно этого действия попадет в операцию присвоения значения переменной y. В результате в переменной х будет число 20, а в переменной у число 25.

Как вы видите, операции в правой части выражения присвоения значения переменной у взяты в скобки, поскольку у операции присвоения может быть только один операнд. Это очень важно учитывать, применяя оператор запятая, поскольку во многих случаях функции принимают только один аргумент, игнорируя все последующие.

Например, в этом примере в консоль будет выведено значение 10, поскольку функция Number() принимает на вход только один аргумент, и это будет результат вычисления выражения x = 10:

let x;
console.log(Number(x = 10, x = x + 20)); // В консоль будет выведено значение 10
Но если мы используем оператор объединения - скобки, и поместим в него оба выражения (x = 10, x = x + 20), то они выполнятся оба, и в функцию Number() попадет результат вычисления последнего выражения: x = x + 20. Поскольку предварительно выполнилось выражение  x = 10, то в результате в консоль будет выведено значение 30.

let x;
console.log(Number((x = 10, x = x + 20))); // В консоль будет выведено значение 30

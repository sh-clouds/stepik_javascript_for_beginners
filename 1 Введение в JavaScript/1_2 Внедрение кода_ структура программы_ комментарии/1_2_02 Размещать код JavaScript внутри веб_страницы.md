Размещать код JavaScript внутри веб-страницы можно в двух местах - внутри заголовка страницы или внутри тела страницы. Эти части страницы обозначены HTML-тегами <HEAD> и <BODY>. (Конечно, код JavaScript можно размещать одновременно и в заголовке и в теле страницы.)


Первый пример - размещение части кода в заголовке страницы и вызов ее из тела страницы.

<html>

<head>
   <script>
        function openWindow() {
           alert("Hello world!");
        }
    </script>
</head>

<body>
    <button type="button" onclick="openWindow()">Open window</button>
</body>

</html>
Мы разместили на странице кнопку с названием "Open window", при нажатии на которую происходит вызов размещенной внутри заголовка функции с названием "openWindow()", при запуске которой выполняется команда JavaScript "alert()", которая открывает на странице всплывающее окно с надписью "Hello world!". 
(Данный пример лишь демонстрирует пример размещения кода, понятие функций будет рассмотрено в следующем модуле.) 

Для проверки вы можете выделить данный кусок примера, сохранить его в файл с любым наименованием и расширением .html и после этого открыть в браузере.

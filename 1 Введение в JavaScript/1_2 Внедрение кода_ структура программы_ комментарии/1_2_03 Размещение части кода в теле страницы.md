Следующий пример - размещение части кода в теле страницы.

<html>
<head>
</head>

<body>
    <button type="button" onclick="openWindow()">Open window</button>

    <script>
        function openWindow() {
            alert("Hello world!");
        }
    </script>
</body>

</html>
В данном случае мы эту же функцию openWindow() разместили в основном теле страницы в самом конце. Такое размещение более предпочтительно при большом количестве JavaScript кода, поскольку большой объем кода в заголовке сильно замедляет загрузку страницы.

Этот пример вы также можете сохранить в .html файл, открыть в браузере и убедиться в том, что код выполняется точно так же.

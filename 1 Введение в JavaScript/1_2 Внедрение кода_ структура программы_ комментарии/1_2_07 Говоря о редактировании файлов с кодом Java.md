Говоря о редактировании файлов с кодом JavaScript необходимо упомянуть различные программные средства, существенно облегчающие и упрощающие этот процесс. В этом курсе у вас не будет необходимости пользования подобными редакторами, поэтому ограничимся поверхностным описанием их возможностей.

Условно все подобные редакторы можно разделить на несколько групп.

Первая это обычные редакторы кода. Наиболее популярные это Sublime и Notepad++.
Они предоставляют достаточно удобный интерфейс, подсветку синтаксиса и еще много удобных возможностей для работы с кодом.

Следующая группа - назовем ее улучшенные редакторы кода, имеют интеграцию с системами контроля версий, средства отладки, системы предложения по коду, возможности рефакторинга и многие другие функции, необходимые для сборки и отладки кода. Такие редакторы более ресурсоемкие, чем обычные редакторы кода, но гораздо более легковесные, чем полноценные среды разработки.
Наиболее популярные это Atom, разработанный компанией GitHub, и Visual Studio Code  от Microsoft.

И последняя группа, это полноценные IDE - среды разработки, покрывающие весь спектр возможностей для удобной и функциональной работы с кодом. К ним относятся в первую очередь продукты компании JetBrains: IntelliJ IDEA, WebStorm, PyCharm, PHPStorm. Microsoft предлагает среду разработки под названием Visual Studio. Все эти системы являются универсальными кроссплатформенными средами разработки, с поддержкой множества языков программирования, включая JavaScript.
